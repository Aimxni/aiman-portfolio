ALU Datapath for Simple RISC Machine

Github Repository

Project Overview

Designed and implemented a SystemVerilog datapath for a Simple RISC Machine, integrating register files, multiplexers, and an ALU to execute eight arithmetic and logical operations. Developed and validated three targeted testbenches, achieving a 100% pass rate against functional specifications. Deployed the design on a DE1-SoC FPGA, using a 7-segment display to showcase real-time computation results with zero runtime errors during demonstration.

Basic Functionality

  • Executes eight arithmetic and logical operations (e.g., ADD, SUB, AND, OR) within a Simple RISC Machine architecture.
  • Integrates register files, multiplexers, and an ALU into a functional datapath for instruction execution.
  • Supports pipeline processing for improved throughput and efficient resource utilization.
  • Processes data from test instructions and displays real-time computation results on a 7-segment display.
  • Validated through comprehensive testbenches, ensuring full compliance with functional specifications before FPGA deployment.

This project was completed as part of the CPEN 211 course at UBC in collaboration with a teammate. I was primarily responsible for designing and implementing the SystemVerilog datapath for the Simple RISC Machine, integrating the register files, multiplexers, and ALU to execute multiple arithmetic and logical operations. Additionally, I created and validated comprehensive testbenches to ensure the design met all functional specifications before deployment on the DE1-SoC FPGA.
