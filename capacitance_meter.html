Capacitance Meter

Github Repository

Project Overview

Designed a microcontroller-based capacitance meter to measure unknown capacitance values in the range of 1nF to 1µF using a 555 timer circuit in astable mode. The ATmega328P microcontroller measured the output frequency from the timer, calculated the corresponding capacitance, and displayed the result on an LCD in real time. Signal timing was captured using internal timers, and all computation and display logic was implemented in embedded C and flashed using CrossIDE. This project was completed as part of ELEC 291 at UBC.

Basic Functionality

  • Measures unknown capacitance in the range of 1nF to 1µF using a 555 timer circuit configured in astable mode.
  • Reads the timer output frequency using the EFM8LB1 microcontroller’s internal timers.
  • Calculates capacitance based on the measured frequency and displays the result on a 16x2 LCD screen.
  • Implements signal measurement, calculation, and display logic entirely in embedded C.
  • Accepts capacitor input through a dedicated test socket, enabling easy swapping during testing.

This project was completed as part of the ELEC 291 course at UBC in collaboration with a teammate. I was primarily responsible for designing the analog front-end circuitry, including the peak detectors, zero-crossing comparators, and input protection circuits. Additionally, I supported the development of the embedded code, contributing to the ADC configuration, phase calculation logic, and LCD display output.
