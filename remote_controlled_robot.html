Remote Controlled Coin Picking Robot

Project Overview

This project involved designing a microcontroller-based autonomous robot capable of detecting, picking up, and counting metallic coins within a defined boundary. Communication between two microcontrollers—PIC32 and EFM8—was established using JDY-40 radio modules, while a Colpitts oscillator enabled real-time metal detection. The robot operated in both manual and automatic modes, with a servo-actuated collection mechanism and an electromagnet to retrieve coins. To enhance functionality, we integrated ultrasonic sensors for obstacle avoidance, LED indicators for movement feedback, a joystick-controlled remote for manual navigation, and a 7-segment display to track collected coins. The result was a stable, responsive, and feature-rich robotic system designed to demonstrate efficient wireless communication, autonomous navigation, and metal detection.

Basic Functionality

  • Metal Detector: Uses a Colpitts oscillator circuit to detect the presence of metallic coins in real time.
  • Perimeter Detector: Combines an ultrasonic sensor to detect physical obstacles and an inductor-based circuit to sense voltage changes, allowing the robot to identify and stay within a defined boundary.
  • Coin Picker: Servo-driven mechanism with an electromagnet to collect and lift detected coins.
  • Automatic and Manual modes: Supports both autonomous operation and joystick-based manual control via wireless communication.

Bonus Features

  • A buzzer that alerts when each state is finished
  • An additional PWM signal is emitted by the microcontroller to power a servo motor to open the door of the oven.
  • A 1-of-4 decoder to indicate each state of the FSM.
  • Cooling fan to accelerate the last state.
  • An additional graph of the temperature read by the digital multimeter.

In addition to the core functionality of the reflow oven controller, we implemented several enhancements to improve usability, feedback, and automation.

LED Status Indicator using Logic Gates

  • To provide a visual representation of the Finite State Machine (FSM) state, we designed a 1-to-4 decoder circuit using NAND and NOT gates, connected to three LEDs.
  • The microcontroller sends either a high (1) or low (0) signal to a BJT, which amplifies the signal before passing it to the decoder.
  • The decoder processes the signal and activates the corresponding LED, providing real-time state indication.

Servo Motor for Oven Door Automation

  • A servo motor was used to automatically open the oven door during the cooling stage.
  • Since the servo motor lacked sufficient torque, we attached a face mask to the oven door to increase tension and facilitate smooth opening.
