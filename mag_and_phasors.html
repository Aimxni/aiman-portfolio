Magnitude and Phasor Voltmeter

Github Repository

Project Overview

This project involved designing a microcontroller-based AC phasor voltmeter to measure and display the RMS voltage and phase difference between two sinusoidal signals. Built around the EFM8LB1 microcontroller, the system used peak detectors and zero-crossing comparators to extract amplitude and phase information, which was then digitized via the built-in ADC. The calculated values were displayed on an LCD screen for real-time monitoring. To ensure accurate timing, timer peripherals were used to measure signal periods and phase shifts, while analog signal conditioning protected inputs from voltage spikes. All data processing and signal analysis were handled in embedded C, compiled and flashed using CrossIDE. I completed this project as a part of ELEC 291 at UBC.

Basic Functionality

  • Measures the RMS voltage of two sinusoidal signals using peak detection and the onboard ADC of the EFM8LB1 microcontroller.
  • Calculates phase difference between reference and test signals via zero-cross detection and timer-based measurement.
  • Displays real-time voltage (RMS), phase angle (in degrees), and signal frequency on a 16x2 LCD screen.
  • Utilizes timer peripherals for accurate synchronization and signal timing analysis.
  • Accepts dual-channel input from function generators, allowing phase-controlled waveform testing.

This project was completed as part of the ELEC 291 course at UBC in collaboration with a teammate. I was primarily responsible for designing the analog front-end circuitry, including the peak detectors, zero-crossing comparators, and input protection circuits. Additionally, I supported the development of the embedded code, contributing to the ADC configuration, phase calculation logic, and LCD display output.
