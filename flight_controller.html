Flight Controller Prototype

Project Overview

This project focused on developing a flight controller prototype implemented on a custom-designed PCB, built around the STM32 microcontroller for high-speed real-time data processing. The system integrated multiple onboard sensors to measure atmospheric pressure, ambient temperature, gyroscopic motion across the X, Y, and Z axes, and altitude in 3D space. Embedded firmware was written to process and log this data continuously into a structured file format, which was stored on an SD card embedded directly into the PCB. To ensure reliable system integration, the board included appropriate power management and signal conditioning circuitry, enabling stable operation in dynamic flight environments.

Basic Functionality

  • Onboard Data Logging: Captured data is stored in a structured file format on an SD card integrated into the PCB, enabling reliable offline analysis and system debugging.
  • Real-Time Data Processing: Powered by the STM32 microcontroller, the system processes sensor inputs in real time to provide immediate flight-related data.
  • Sensor Data Acquisition: Uses onboard sensors to measure atmospheric pressure, ambient temperature, gyroscopic data across X, Y, and Z axes, and altitude in 3D space for accurate environmental and motion tracking.
