Real Time Speech Recognition System

Project Overview

This project focused on designing a real-time speech recognition system implemented on a custom-designed PCB, centered around the high-performance STM32H750VBT6 microcontroller. The system utilized the INMP441 digital MEMS microphone to capture audio signals with precision, which were then processed on board for keyword recognition and command classification. A proprietary USB Type-C connector was integrated into the PCB to enable reliable interfacing with external devices, with appropriate ESD protection circuitry to ensure robust operation in various environments.

Basic Functionality

  • Real-Time Speech Recognition: Utilizes the STM32H750VBT6 microcontroller to capture and process audio data in real time, enabling accurate keyword spotting and voice command recognition.
  • High-Precision Audio Input: Features the INMP441 digital MEMS microphone for high-fidelity audio capture, directly interfaced via I2S for efficient data transfer.
  • USB Type-C Interface: Includes a proprietary USB Type-C connector for power and data communication with external devices such as PCs, power banks, or development boards.

Bonus Features

  • Ultrasonic sensor that allows the robot to avoid obstacles on its way.
  • Rear LEDs that indicate which way the robot is rotating or moving backward.
  • 7-segment display that shows how many coins have been picked up.

In addition to the core functionality of the Remote Controlled Coin Picking Robot, we implemented several enhancements to improve usability, feedback, and automation.

Coin Counter Display

  • Integrated a 7-segment display to show the number of coins picked up by the car.
  • Each time the servo motor activates to pick up a coin, the system increments a counter, and it is played in real time on the 7-segment display.
  • The count increments by one with each successful coin pickup, and once the count reaches 20, it resets to zero and begins counting again, following the coin collection requirements defined in the project scope.

Tail Light Direction Indicators

  • LEDs are positioned to mimic the conventional tail lights found in standard vehicles.
  • The left or right LED lights up to indicate the direction of movement (left or right turn).
  • When the car reverses, both LEDSs follow the standard automotive convention for indicating reverse motion.
  • This enhanced the realism and user experience by simulating actual car lighting behaviour.
  • Provides visual feedback to easily determine the direction in which the car is moving.
